Optimizing the CSS and Javascript
External resources can be optimized automatically, which can reduce both the size and number of requests made to your website.

1) Browse to Configuration -> Development -> Performance Interface
2) Scroll down to Bandwidth Optimization
3) Check "Aggregate and compress CSS files"
4) Check "Aggregate JavaScript files."
5) Click "Save configuration"
6) Open up site in Firebug and view the script and styles references
