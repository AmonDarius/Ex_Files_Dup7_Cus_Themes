How to install and use Admin Menu module for enhanced functionality and navigation and disabling default toolbar

http://drupal.org/project/admin_menu


Directions
-------------------

Disabling Toolbar Module
1) From Modules interface locate "Toolbar" module
2) Uncheck "Toolbar" module
3) Click "Save Configuration" button
4) Choose "Uninstall tab"
5) Check "UNINSTALL" option
6) Click "Uninstall" button
7) Confirm Uninstall by clicking the "Uninstall" button again
8) Click on "List" tab to go back to Module list
9) Confirm "Toolbar" menu at top of page is gone

Downloading, installing and enabling Admin Menu module
1) Browse to http://drupal.org/project/admin_menu
2) Download the latest Drupal 7 version of module
3) Browse to http://samoca:8082/admin/modules
4) Choose "Install new module" link
5) Choose Browse from the "Upload a module or theme archive to install"
6) Select the admin_menu module file you downloaded
7) Click the "Install" button
8) Click "Enable newly added modules"
9) Scroll down to "Administration" fieldset and check "Administration menu"
10)Click "Save Configuration" button
11)Confirm new "Admin Toolbar" menu at top of page.

Setting Permissions to Admin Menu Module
1) From Modules interface, click "Permissions" next to "Admin Menu"
2) Confirm that "Administrator" role has permissions to use Admin Menu module

Configuring Admin Menu Module
1) Browse to http://samoca:8082/admin/config/administration/admin_menu
2) Review various configuration options
3) Based on design or theme being created or installed you may want to adjust the top margin of the Admin Menu by selecting the "Adjust top margin" option.
4) Modify settings as needed and Click "Save configuration: button
